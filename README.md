# CertFlow

CertFlow is a single-page React app for tracking certification study plans, courses, lab progress, writeups, and pomodoro sessions.

## Stack

- React
- Vite

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The current app code was imported from the standalone `certflow-v5.jsx` file.
- Seed data is embedded in the UI component, so the app starts with demo content.
